# 正文命名规范

## 文件命名规则

```
第{章号}章-{星球名或章节标题}.txt
```

### 示例

```
第01章-代码宇宙的第一条命.txt
第02章-修仙世界的一百种死法.txt
第03章-末日废土的抽象生存指南.txt
```

## 文件存放位置

所有正文文件统一存放在 `正文全稿/` 目录下，不分子目录。

## 内容格式

- 纯文本（.txt 格式）
- 不包含标题行，从正文第一句开始
- 每章独立文件，不合并

## 元数据

每章正文文件开头不写元数据。元数据统一存放在：
- **剧情方案：** `剧情推演方案.md`（创作过程中保留）
- **状态记录：** 由「故事线与人物线更新」技能自动维护

## 注意事项

- 文件名中的章号统一使用两位数字（01, 02, 03…）
- 星球名或章节标题使用中文，不加标点符号
- 禁止使用空格和特殊字符
- 修改正文后需同步更新版本备份
