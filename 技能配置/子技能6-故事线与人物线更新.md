# 子技能6：故事线与人物线更新

## 适用场景
每章创作完成后，自动更新世界观文档、人物状态文档、伏笔追踪表，确保故事线始终一致可追踪。

## 输入格式

```
【本章正文】
（最终定稿版正文）

【前文状态报告】
（来自子技能1，用于对比状态变化）

【设定文档目录路径】
（大纲细纲/ 设定文档/ 等目录路径）
```

## 输出格式

```
【更新报告】
✅ 故事线已更新
✅ 人物状态已更新
✅ 伏笔表已更新
✅ 世界观文档已更新

【更新详情】
📖 故事线：
  - 新增内容概要：[简述]
  - 当前总章节数：[X]

👤 人物状态变更：
  - 和意味：[状态变化详情]
  - 抽象系统：[状态变化详情]
  - [其他人物]：[状态变化详情]

🔗 伏笔状态：
  - 新埋设：[伏笔内容 → 当前章号]
  - 已回收：[伏笔内容 → 回收章号]
  - 持续追踪：[伏笔内容 → 状态]

📋 世界观更新：
  - 新增设定：[设定项]
  - 已更新设定：[设定项+变更内容]
```

## 执行步骤

1. 读取本章最终正文，提取关键剧情节点和人物变化
2. 更新故事线文档：以追加方式写入本章剧情摘要
3. 更新人物状态文档：逐人记录状态变化
4. 更新伏笔追踪表：标记新埋设的伏笔、已回收的伏笔
5. 更新世界观文档：补充新出现的设定元素
6. 输出更新报告，确认所有更新完成

## 注意事项

- 所有更新采用追加方式，不覆盖已有内容，保留完整历史
- 所有更新必须基于正文内容，不凭空添加
- 伏笔状态更新后，区分「活跃中」「已回收」「已过期」
- 每次更新后保留旧版本快照，至少保留最近5次更新的历史
