# 子技能5：内容优化与打磨

## 适用场景
正文生成和合规审查后，对内容进行润色优化——润色对话、增强画面感、调整叙事节奏。

## 输入格式

```
【待优化正文】
（正文全文）

【审查报告】（可选）
（来自合规与一致性审查的输出，标注了待优化位置）
```

## 输出格式

```
【优化后正文】
（润色打磨后的完整正文）

【优化日志】
- 修改对话数：[X]
- 增强画面感段落数：[X]
- 调整节奏段落数：[X]
- 其他优化：[具体列表]
```

## 执行步骤

1. 通读全文，标记需要优化的对话部分
2. 优化人物对话：使其更符合角色性格，增加口语化和人味儿
3. 增强画面感：补充环境描写、动作细节、感官体验（视觉/听觉/触觉）
4. 调整叙事节奏：精简拖沓段落，补充必要的细节描写
5. 检查过渡衔接：确保段落之间自然流畅
6. 对照审查报告（如有），逐条处理优化建议
7. 输出优化后的正文及优化日志

## 注意事项

- 不改变原剧情走向和核心设定，不增减关键信息
- 只优化表达方式，不改变对话内容含义
- 保持正文的自然语言风格，避免过度修饰雕琢
- 画面感增强要以「展示」代替「讲述」（show, don't tell）
- 节奏调整以「读起来不累、不拖、不跳」为标准
