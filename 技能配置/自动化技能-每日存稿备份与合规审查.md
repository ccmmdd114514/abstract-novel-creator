# 自动化技能：每日存稿备份与合规审查

## 适用场景
每日定时自动执行，扫描当日新增/修改的存稿，自动备份至版本备份目录，同时扫描全文检测敏感词和违规风险，无需人工干预。

## 触发方式

- 每日定时触发（推荐每日创作结束后自动执行）
- 也可在每章定稿后手动触发

## 输入

```
【项目工作空间路径】
（网文项目根目录）

【扫描范围】
- 正文全稿目录
- 大纲细纲目录
- 设定文档目录
```

## 输出

```
【每日执行报告】
📦 备份情况：
  - 备份文件数：[X]
  - 新增/修改文件：[文件列表]
  - 备份存放位置：[路径]

🔍 合规扫描结果：
  - 扫描文件数：[X]
  - 发现风险：[是/否]
  - 风险等级：[P0严重 / P1轻微 / 无风险]
  - 问题详情：[如发现问题，列出位置+类型]

✅ 执行结论：[正常完成 / 发现风险需处理]
```

## 执行步骤

1. 扫描正文全稿目录，识别当日新增或修改的文件（对比版本历史记录）
2. 将新增/修改的文件完整复制至版本备份目录
3. 更新版本历史记录文件
4. 扫描所有正文文件，检测敏感词和违规风险（敏感词库可配置）
5. 如发现P0级别风险，立即标记并生成问题报告
6. 输出每日执行报告

## 注意事项

- 自动执行，无需人工干预
- 敏感词库可配置更新（建议每月同步一次平台最新审核规则）
- P0级别风险需立即通知作者处理
- 每日报告保留最近30天记录
- 如当日无新增或修改文件，仅做合规扫描并输出「无变更」
