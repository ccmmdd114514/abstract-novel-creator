# 子技能7：自动版本备份

## 适用场景
每章定稿后，将本章所有相关产出文件完整备份至版本备份目录，保留完整版本历史。

## 输入格式

```
【备份源文件清单】
- 本章正文路径：[路径]
- 本章剧情方案路径：[路径]
- 当前设定文档路径：[路径]
- 当前故事线状态路径：[路径]

【版本号】
（格式：v{major}.{minor}，每次主备份 major+1）
```

## 输出格式

```
【备份报告】
✅ 备份完成

【备份信息】
- 备份路径：[路径]
- 备份时间：[时间戳]
- 版本号：[版本号]
- 包含文件数：[X]
- 总大小：[X KB]
- 版本历史记录：[文件路径]
```

## 执行步骤

1. 收集本章所有产出文件（正文、剧情方案、审查报告、更新后的设定文档等）
2. 在版本备份目录下按版本号创建子目录（如 v01/ v02/ v03/）
3. 将所有文件完整复制至版本子目录（复制非移动，原文件保留）
4. 更新版本历史记录文件（追加本次备份信息）
5. 验证备份完整性：核对文件数量、文件大小
6. 输出备份报告

## 注意事项

- 完整复制而非移动，原目录文件保留
- 版本号自动递增，保留所有历史版本
- 每次备份至少包含：正文、剧情方案、当前设定文档、故事线状态
- 备份后不可修改备份目录内的文件
- 建议每5个版本或每周清理一次30天前的旧版本（可配置）
